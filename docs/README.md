# Документация проекта

Сводный указатель для навигации по документации системы анализа вовлечённости при просмотре образовательного видеоконтента.

> Корневой [README.md](../README.md) даёт краткое введение (EN), Quick Start и обзор архитектуры. Здесь собрана подробная техническая документация по всем разделам проекта.

---

## Разделы

### Серверная часть

[`backend/`](backend/README.md) – архитектура, разработка, развёртывание, переменные окружения, формат API, OpenAPI-спецификация.

### ML-пайплайн

[`pipeline/`](pipeline/README.md) – обзор пайплайна обработки, детекция лиц (MediaPipe), распознавание эмоций, Eye Aspect Ratio, Head Pose Estimation, временное сглаживание метрик системы.

### Расчёт вовлечённости

[`engagement-calculation/`](engagement-calculation/README.md) – итоговая формула и веса компонентов, модификаторы, адаптивное сглаживание, классификация уровней вовлечённости и определение тренда, научные публикации, задействованные в проекте.

### Фронтенд

[`frontend/`](frontend/README.md) – Streamlit-приложения для демонстрации работы системы и настройки параметров (`engagement_app.py`, `video_engagement_app.py`, `tools/param_testing_app.py`), переменные окружения, ограничения и тонкости при работе с разработанным фронтендом.
